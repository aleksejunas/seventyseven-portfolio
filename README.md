# Welcome to the seventyseven-portfolio project

## A portfolio project for the seventyseven company

It is a fullstack project that uses the following technologies:

Vite, React, TypeScript, TailwindCSS, Sanity, and more.

### For Developers

1. Clone the repository
2. Run `pnpm install`
3. Run the Frontend dev server with `pnpm dev`
4. Run the Sanity Backend Server with `pnpm sanity`
5. Go to `http://localhost:5173` to see the frontend
6. Go to `http://localhost:3333` to see the Sanity Studio
